# instagram-openai-followers-analysis
Built a real-world Instagram data analysis project using pure Python.  Cleaned messy social-media data, normalized K/M metrics, handled missing values,  and answered analytical questions like top posts, followers, and categories.

## Problem Statement
Analyze raw Instagram data of OpenAI followers and answer key analytical questions.

## Questions Answered
- Who has the maximum posts
- Who has the maximum followers
- Who follows the maximum people
- How many page categories exist and how many users per category

## Tech Stack
- Python
- JSON
- Pure Python data processing (no pandas)

## Key Learnings
- Data cleaning & validation
- Handling K/M numeric formats
- Defensive programming
- Real-world data inconsistencies
- Instagram-style data parsing
